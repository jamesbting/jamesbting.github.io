Sudoku Game

December 2019 - Present

I created a Sudoku game in Python, which includes a GUI, automatic solver, stopwatch and a game generator.

The GUI and the stopwatch were built using the Tkinter module. The solver employed a recursive backtracking algorithim to test possible solutions to the board, and then eliminate those that would not lead to a valid soution.

Login System

November 2019 - Present

I created a login system in Python, which includes a GUI, a database to store all the usernames, and lets users create accounts, change passwords, and login.

This project was my first personal project I had ever created. I had planned it to be a simple program that let me practice file IO and OOD in Python, but would not include a GUI and the user was expected to interact with the console.

Upon learning about cryptographic hash functions, I had wanted to apply that to the login system, and be done with it. In the end, I added a GUI (built using Tkinter) a SHA-256 encryption algorithim, and a database to store all the account informations.
