League of Legends Match Outcome prediction with Deep Learning

Deep Learning, PyTorch, Riot API

May 2020 - Present

I built a Binary Classification Neural Network using the PyTorch library, and trained it to predict the outcomes of League of Legends games, based on each participant's stats. I achieved approximately 92% validation accuracy, on a dataset of approximately 3403 matches.

The dataset was created by me using Riot's API and the RiotWatcher wrapper library. The script that collects the data starts with a match, then, for a random summoner in that match, pulls the past match history and then collects the data of that match, and continues the process recursively

Sudoku Game

Backtracking Algorithms, GUIs

December 2019

I created a Sudoku game in Python, which includes a GUI, automatic solver, stopwatch and a game generator.

The GUI and the stopwatch were built using the Tkinter module. The solver employed a recursive backtracking algorithm to test possible solutions to the board, and then eliminate those that would not lead to a valid solution. This project helped me learn about more types of algorithms beyond those that are typically covered in the classroom, but are nevertheless useful tools.

Login System

Cybersecurity, Databases, GUIs

November 2019 - January 2020

I created a login system in Python, which includes a GUI, a database to store all the usernames, and lets users create accounts, change passwords, and login.

This project was my first personal project I had ever created. I had planned it to be a simple program that let me practice file IO and OOD in Python, but would not include a GUI and the user was expected to interact with the console.

Upon learning about cryptographic hash functions, I had wanted to apply that to the login system, and be done with it. In the end, I added a GUI (built using Tkinter) a SHA-256 encryption algorithm, and a SQLite3 database to store all the account information.
